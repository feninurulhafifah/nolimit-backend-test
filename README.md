1. Clone repositori tersebut ke lokal.
2. Jalankan npm install untuk memasang semua dependensi.
3. Buat database kosong bernama nolimit_blog di MySQL local.
4. Salin file .env.example menjadi .env lalu sesuaikan kredensial database (host, user, pass, port) di lokal Anda.
5. Jalankan perintah "npm run dev" untuk menjalankan server (tabel database akan otomatis dibuat oleh Sequelize).
6. Jalankan perintah "npm test" untuk menjalankan seluruh rangkaian unit testing.
