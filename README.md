## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/feninurulhafifah/nolimit-backend-test
   cd nolimit-backend-test
3. Install dependencies
   ```bash
   npm install
4. Copy the .env.example file to .env
   ```bash
   cp .env.example .env
5. Run the Application
   ```bash
   npm run dev
6. Run unit test
   ```bash
   npm test
